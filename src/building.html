Як будуються веб-сторінки

  • HTML

    Веб-сторінка будується на основі мови гіпертекстової розмітки. Офіційна назва мови HTML (Hyper Text Markup Language).

  • CSS

    Для відображення вмісту на веб-сторінці створені та служать каскадні таблиці стилів, інакше CSS.

  • JAVASCRIPT

    У створенні динамічних сторінок на допомогу розробникам «приходять» мови скриптів. Найпопулярніша мова JavaScript.

HTML
Мова гіпертекстової розмітки сторінки. Вона використовується для того, щоб дати браузеру зрозуміти, як потрібно відображати завантажений сайт.
CSS
Формальна мова опису зовнішнього вигляду веб-сторінки, написаного з використанням мови розмітки (найчастіше HTML або XHTML).
JAVASCRIPT
Мультипарадигменна мова програмування. Підтримує об'єктно-орієнтований, імперативний та функціональний стилі.
